# Jobs

A modern job board & community platform UI prototype built with SvelteKit 5.

## Tech Stack

- **Framework:** SvelteKit 5, Svelte 5 (runes)
- **UI:** shadcn-svelte, bits-ui, Lucide icons
- **Styling:** Tailwind CSS v4
- **AI:** Vercel AI SDK + Anthropic

## Features

- 8 color themes (Zinc, Cursor, Catppuccin, Midnight, Cupertino, JetBrains, Airbnb, Neobrutalism)
- LinkedIn-style job board with filters
- TikTok-style video feed
- Tinder-style developer matching
- Google Maps integration
- AI chat interface
- Community feed with articles

## Getting Started

```bash
npm install
npm run dev
```
